Celestia Node API Docs

This is the Celestia Node API Docs reference page for being able to make API requests to your Celestia Node.

In order to query the API, you will need to setup your node. Resources on how to do this can be found in the following guide.

State endpoints‚Äã

GET /balance‚Äã

Returns the balance of the default account address of the node.

Request

curl -X GET http://<ip>:26658/balance

Response

{
   "denom":"uceles",
   "amount":"999995000000000"
}

GET /balance/{address}‚Äã

Returns the balance of the given address.

Request

curl -X GET http://<ip>:26658/balance/celes1vuw7pdcyap2u62x6dyqjdzznlg25jll6jj76d0

Response

{
   "denom":"uceles",
   "amount":"999995000000000"
}
